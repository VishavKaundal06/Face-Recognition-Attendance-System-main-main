# Smart Attendance Backend (JavaScript)

Unified Node.js/Express backend for smart attendance system with:
- Face recognition (face-api.js or opencv4nodejs)
- User management
- Attendance tracking
- CSV/Excel/PDF reporting
- File uploads
- Static file serving
- WebSocket support for real-time events

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm run dev
   ```

## Project Structure
- `src/` — Main source code
- `uploads/` — Uploaded images/files
- `reports/` — Generated reports
